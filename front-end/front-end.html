Some help:

Search and Translate: Requires an input term. Translate is another way of searching but takes a less literal sense of the term.
Endpoint: After an image is earched, a large ID is returned with the picture. Input the id into the search box and the gif of that id will be returned.
Search Sticker: Requires an input term and returns the sticker of the term, may or may not be different than Search and/or Translate.
Fun Facts: Click on the image to get rid of it and to clear all Endpoints and Gifs Click "Clear All :D"
